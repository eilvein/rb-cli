# rb-cli

A simple ROOBO CLI.

### Installation

```
yarn install -g rb-cli
```

### Usage

```
rb <command> [options]
# help
rb --help
```

### Generate Project

Generate project using template.

```
rb create <name>
```

The current template

-   gulp webpack project template
-   webpack project template
-   vue project template
